💰 FinControl

Aplicativo mobile para controle financeiro pessoal, com foco em simplicidade, visualização de dados e acessibilidade para idosos.

🚀 Sobre o Projeto

O FinControl é um aplicativo desenvolvido para auxiliar usuários no gerenciamento de suas finanças pessoais de forma prática e intuitiva.

A aplicação permite registrar, editar e remover movimentações financeiras, além de oferecer uma visualização clara do saldo e da distribuição de gastos.

✨ Funcionalidades
🔐 Autenticação

Cadastro de usuário (nome, e-mail e senha)

Login seguro

Aceite de termos de privacidade

🏠 Dashboard (Home)

Exibição do saldo total

Gráfico em pizza com distribuição dos gastos

Botão de acesso rápido para adicionar registros

➕ Gerenciamento de Registros

Cadastro de:

💰 Receitas

💸 Despesas

Campos:

Valor

Categoria

Descrição

✏️ Edição e Exclusão

Editar registros existentes

Remover movimentações

Correção de dados de forma simples e rápida

📋 Registros (Lançamentos)

Listagem completa das movimentações

Identificação visual de receitas e despesas

Exibição de valor, tipo e descrição

📊 Extrato

Exibição das últimas movimentações

Indicadores visuais (+ / -)

Foco em leitura rápida

🧭 Navegação

O aplicativo utiliza uma barra inferior com acesso direto a:

🏠 Home

📊 Extrato

➕ Adicionar
